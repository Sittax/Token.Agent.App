# Token.Agent.App

Repositório para armazenar as releases do Sittax Token.

Este repositório contém os instaladores e artefatos relacionados às versões do Sittax Token. O objetivo é centralizar os arquivos de release (MSI, instaladores, changelogs) para fácil acesso.

Release atual
- Versão: 3.0.0
- Arquivo esperado: `Sittax Token.msi`

Como publicamos releases
1. É recomendado criar uma Release no GitHub (menu "Releases") para a versão (ex.: `v3.0.0`) e anexar o MSI como asset — assim o instalador fica disponível para download simples.
2. Alternativamente, podemos commitar o MSI diretamente no repositório em uma pasta `releases/v3.0.0/`, porém arquivos binários grandes não são ideais no histórico do Git.
